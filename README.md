## ✨ 工具箱介绍
本工具是专为安卓Termux免Root终端打造的OpenClaw（小龙虾）全自动化部署管理脚本，采用Linux Shell编写，界面炫酷带彩色终端样式、ASCII艺术横幅、实时安装进度条，集环境搭建、一键部署、服务启停、配置修改、命令查询于一体，零基础也可快速上手部署运行AI小龙虾网关服务。
 
修复交互式输入兼容性问题，全程静默后台安装，自动处理依赖缺失、镜像源切换、进程守护、会话隔离，稳定适配手机移动端Linux运行场景。
